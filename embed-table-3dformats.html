Proprietäre Formate (Softwaregebunden)

  • Rhino Files (*.3dm).
  • Autodesk Inventor Files (.ipt,.iam - up to Inventor 2019)
  • SolidWorks Files (.prt,.asm, .sldprt,.sldasm , *.slddrw)
  • Autodesk Alias (*.wire)
  • 123D File (*.123dx)
  • AutoCAD DWG Files (*.dwg)
  • SketchUp Files (*.skp)
  • CATIAV5 Files (.CATProduct,.CATPart)
  • Autodesk Fusion 360 Archive Files (*.f3d)
  • Siemens NX (*prt)
  • Parasolid Files (.x_b,.x_t)
  • Pro/ENGINEER and Creo Parametric Files (.asm,.prt)
  • Pro/ENGINEER Granite Files (*.g)
  • Pro/ENGINEER Neutral Files(*.neu)
  • Autodesk Fusion 360 Toolpath Archive Files (*.cam360)
  • Blender File (.blend)
